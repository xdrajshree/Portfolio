# Raj Shree Portfolio

Black and gold portfolio website for Raj Shree, IIT Madras BS Degree Data Science student.

## Edit Before Deploying

- Replace `your-email@example.com` in `index.html` with your real email.
- Replace the `#` links for GitHub, LinkedIn, and project live links.
- Update project names, descriptions, and languages as your real-time projects go live.

## Run Locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this folder to GitHub.
2. Open Vercel and import the GitHub repository.
3. Vercel should detect Vite automatically.
4. Use these settings if needed:
   - Build command: `npm run build`
   - Output directory: `dist`
5. Add your custom domain from the Vercel project dashboard.
